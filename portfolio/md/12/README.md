```markdown
# クイズアプリ

FastAPIをバックエンド、Streamlitをフロントエンドに使用したシンプルなクイズアプリです。バックエンドはクイズデータを提供し、回答の正誤判定を行います。フロントエンドはクイズを表示し、ユーザーからの回答を受け付け、結果を表示します。

## 工夫点

*   **CORSミドルウェアの設定:** 異なるオリジン（localhost:8501）からのリクエストを許可するために、CORSミドルウェアを設定しています。

*   **Streamlitのsession_stateの活用:** クイズの進行状況、スコア、回答履歴を保持するために、Streamlitの`session_state`を使用しています。これにより、ページリロード時に状態がリセットされるのを防ぎます。

*   **Pydantic BaseModelによるデータモデル定義:** FastAPIのデータモデル(Pydantic BaseModel)を用いて、APIの入出力の型を定義しています。これにより、データの整合性を保ち、validationを容易にしています。
```
